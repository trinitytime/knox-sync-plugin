# Knox Sync

[English](./README.md) | 한국어

Obsidian의 문서와 파일을 Knox portal를 이용해서 동기화하는 플러그인입니다.

## 기능

- Knox을 백업 저장소로 사용하여 기기간 동기화
- Knox의 공유 기능을 이용하여 사용자간 문서 공유
- 일정 시간마다 동기화
- 최종 수정 후 일정 시간 수정이 없으면 동기화
- 한달 마다 Knox의 문서를 갱신해서 문서 삭제 방지

## 사용법

1. Knox portal의 "할 일"에서 신규 프로젝트를 생성
   이때 이름을 +로 시작해야 한다. 예) +옵시디언, +친구
2. Knox 환경 설정 -> 기본 설정 -> 다른 브라우저 자동 로그인 -> 사용
3. 플러그인을 vault의 플러그인 폴더에 설치
4. 명령어 팔레트에서 "sync start" 선택

- 용량 정책이 명확하지 않아 파일 크기가 너무 큰 경우 동기화가 안될 수도 있습니다.
- 문서 충돌 발생시 마지막 수정된 문서로 덮어 씌워집니다. (Last Write Wins)
